# Cognitive Neuroscience Domain Map

Use this file to generalize beyond one EEG/ERP microstate project. Identify the user's domain before choosing literature, methods, and interpretation language.

## Major Cognitive Domains

### Attention

Typical questions:

- selective attention;
- sustained attention;
- orienting;
- attentional control;
- distractor suppression;
- vigilance and lapses.

Common methods:

- cueing tasks, CPT, oddball, visual search, Flanker, Stroop;
- EEG/ERP: P1/N1, N2pc, P3, CNV, alpha modulation;
- fMRI/fNIRS: dorsal/ventral attention networks, frontoparietal control.

Interpretation caution:

- attention effects often overlap with arousal, task engagement, perception, and response demands.

### Executive Function And Cognitive Control

Typical questions:

- inhibition;
- conflict monitoring;
- task switching;
- working memory updating;
- cognitive flexibility;
- error monitoring.

Common methods:

- Go/No-Go, stop-signal, Flanker, Stroop, task switching, n-back;
- EEG/ERP: N2, P3, ERN/Pe, CNV, frontal theta;
- fMRI: ACC, pre-SMA, inferior frontal cortex, basal ganglia, frontoparietal control network.

Interpretation caution:

- executive tasks are rarely process-pure. Always separate task demand from inferred mechanism.

### Memory And Learning

Typical questions:

- encoding;
- consolidation;
- retrieval;
- recognition;
- prediction;
- statistical learning;
- reinforcement learning.

Common methods:

- recognition/recall tasks, associative learning, sequence learning, reward learning;
- EEG/ERP: old/new effect, FN400, LPC, feedback-related negativity;
- fMRI: hippocampus, medial temporal lobe, prefrontal memory control;
- computational modeling: prediction error, reinforcement learning, drift diffusion.

Interpretation caution:

- behavioral accuracy may hide strategy differences; model-based or process-level analysis may be needed.

### Language And Reading

Typical questions:

- phonology;
- orthography;
- semantics;
- syntax;
- sentence processing;
- reading fluency;
- dyslexia and reading risk.

Common methods:

- lexical decision, naming, sentence reading, naturalistic reading, eye-tracking;
- EEG/ERP: MMN, N170, N250, N400, P600, LPC;
- fMRI: left temporal, occipitotemporal, inferior frontal language regions.

Interpretation caution:

- language tasks mix perception, attention, memory, and motor response. Developmental and language-specific factors matter.

### Numerical Cognition And Mathematics

Typical questions:

- number sense;
- symbolic/non-symbolic magnitude;
- arithmetic;
- mathematical learning;
- dyscalculia and math risk;
- working memory and executive-function contributions.

Common methods:

- magnitude comparison, arithmetic verification, number line, working-memory tasks;
- EEG/ERP: N1/P2, N2, P3, late positivities depending on task;
- fMRI/fNIRS: intraparietal sulcus, frontal control regions.

Interpretation caution:

- math risk may reflect domain-specific numerical processing, domain-general executive function, language, attention, or instruction history.

### Emotion And Social Cognition

Typical questions:

- emotion perception;
- reward/punishment;
- social evaluation;
- mentalizing;
- self-other distinction;
- empathy.

Common methods:

- face/emotion tasks, reward tasks, trust games, social feedback tasks;
- EEG/ERP: EPN, LPP, FRN/RewP, N170, P3;
- fMRI: amygdala, striatum, mPFC, TPJ.

Interpretation caution:

- affective salience, attention, arousal, and task relevance are hard to separate.

## Method Families

### EEG/ERP/MEG

Strengths:

- millisecond timing;
- good for processing stages;
- useful for developmental and task dynamics.

Risks:

- source localization ambiguity;
- reference and preprocessing sensitivity;
- component overlap;
- multiple comparisons;
- overinterpreting scalp topography.

### fMRI/fNIRS

Strengths:

- spatial localization and network interpretation;
- useful for systems-level models.

Risks:

- temporal sluggishness;
- motion artifacts, especially in children;
- reverse inference;
- multiple comparisons and circular ROI selection.

### Eye-Tracking And Behavior

Strengths:

- direct task-performance and online processing measures;
- strong complement to neural data.

Risks:

- behavior may be multiply determined;
- eye movements reflect task strategy as well as cognition.

### Computational Modeling

Strengths:

- separates latent processes such as drift rate, threshold, bias, prediction error, or learning rate.

Risks:

- model identifiability;
- parameter recovery;
- overfitting;
- interpreting parameters without validation.

## Mentor Rule

Never force a result into a familiar component, brain region, or theory. First ask whether the task, population, method, and timing actually support that interpretation.
