# Project Setup Guide

Use this guide when adapting the mentor to a new cognitive neuroscience project.

## Recommended Folder Structure

```text
project/
  AGENTS.md
  00_project_context/
  01_materials/
  02_results/
  03_literature/
  04_manuscript/
  05_code/
```

## Core Orientation Files

Create these files when possible:

- `AGENTS.md`: project-specific mentor behavior, topic, claims, and local rules.
- `00_project_context/project_profile.md`: study title, research questions, sample, task, methods, variables, hypotheses.
- `00_project_context/organized_workspace_manifest.md`: where materials are stored and how to use them.
- `03_literature/literature_inventory_by_theme.md`: local literature organized by theme.
- `03_literature/mentor_literature_use_guide.md`: when to use each literature category.
- `02_results/result_interpretation_template.md`: standard structure for result interpretation.

## Mentor Rule

Use local project files for user-specific evidence, but search external literature when the local evidence is not enough. Never treat a project folder as a closed world.
