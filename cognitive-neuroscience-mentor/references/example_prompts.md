# Example Prompts

Use these examples to help users invoke the skill effectively.

## Result Interpretation

```text
Use cognitive-neuroscience-mentor in standard mode. Interpret this ERP topomap result. The task is Go/No-Go, groups are TD and attention-risk, and the main difference appears from 300-600 ms.
```

```text
请作为认知神经科学导师解释这个 fMRI ROI 结果。我的任务是 working memory n-back，DLPFC activation 与反应时相关，但组间差异不显著。
```

## Literature Search

```text
帮我查找并综合文献：儿童数学困难是否与 inhibitory control 或 working memory 有关？请区分直接证据、间接证据和背景证据。
```

```text
Find recent peer-reviewed literature on EEG markers of sustained attention in children. I need sources suitable for an Introduction section.
```

## Study Design

```text
Use cognitive-neuroscience-mentor to refine my research question. I want to study language prediction using EEG, but my question is still broad.
```

```text
请帮我设计一个 fNIRS 儿童执行功能研究，重点检查任务、变量、样本量风险和伦理边界。
```

## Statistics And Code Review

```text
请检查这个 mixed-effects model 是否适合我的 ERP amplitude 数据。被试是随机效应，条件是 within-subject，组别是 between-subject。
```

```text
Review this preprocessing script scientifically, not just technically. Focus on filtering, baseline, artifact rejection, trial counts, and condition coding.
```

## Manuscript Writing

```text
Use full mode to rebuild my Discussion. First create controlling motivation, claim register, and section blueprint, then draft.
```

```text
请按 Nature-style 的 claim-evidence-boundary 逻辑帮我重写 abstract，不要夸大结果。
```

## Reviewer Response

```text
Reviewer says our ERP interpretation is overclaimed. Help me classify the comment, decide whether to agree, and draft a response with revised manuscript wording.
```
