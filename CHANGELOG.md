# Changelog

## v0.3.1

- Updated `mentor-anything` to require visible literature support for result interpretation, mechanism claims, manuscript wording, novelty, reviewer-facing, and grant/application claims.
- Added evidence levels to mentor-anything answers: direct, partial, background, contradictory, and unverified.

## v0.3.0

- Added `mentor-anything` as the default expert Q&A subfunction.
- Added a protocol for classifying any cognitive neuroscience question and answering with expert reasoning, boundaries, uncertainty, and next steps.

## v0.2.0

- Generalized the skill beyond EEG/ERP microstates to broad cognitive neuroscience.
- Added quick, standard, and full work modes.
- Added project setup guidance.
- Added ethics and interpretation boundaries.
- Added example prompts.
- Expanded scientific writing workflow with claim register, section blueprint, and writing rationale matrix.

## v0.1.0

- Initial cognitive neuroscience mentor skill.
- Added research workflow, ERP/microstate guidance, statistics guide, and manuscript review guide.
